# Changelog

## 0.1.0

- Initial offline-first FastAPI backend.
- React/Vite/Tailwind frontend with dashboard, upload, search, history, analytics, settings, and about pages.
- SQLite persistence with files, processed data, history, and settings tables.
- Local CPU pipeline adapters for PyMuPDF, python-docx, OpenCV, PaddleOCR, whisper.cpp, and llama.cpp.
- JSON and CSV export.
- Docker, GitLab CI/CD, pre-commit, tests, and documentation.
