# User Guide

1. Open the web app.
2. Go to Upload.
3. Drag a PDF, image, audio file, DOCX, or text file onto the upload area.
4. Wait for local processing stages to complete.
5. Review the structured JSON.
6. Use Search to find records by title, entity, keyword, category, or date.
7. Use Settings to export JSON or CSV.

The app is designed to keep working when Wi-Fi is off after dependencies and local model files are installed.
